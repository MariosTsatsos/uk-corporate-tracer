# docs/

Place `graph_preview.png` here — a screenshot of the network graph generated by `build_graph.py`.

**How to generate:**
1. Run `python build_graph.py` (requires a populated `asset_discovery.db`)
2. Open `output/corporate_network.html` in your browser
3. Let the physics stabilise, zoom out to fit all nodes
4. Screenshot and save as `docs/graph_preview.png`
